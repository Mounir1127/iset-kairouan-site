<!-- Instructions pour ajouter le logo ISET Kairouan -->

Pour utiliser le logo officiel de l'ISET Kairouan :

1. Téléchargez le logo officiel depuis : https://www.isetkr.rnu.tn/images/logo.png
   
2. Placez le fichier dans : src/assets/images/logo-iset-kairouan.png

3. Si vous voulez utiliser le logo local au lieu de l'URL externe, 
   remplacez dans app.component.html :
   
   src="https://www.isetkr.rnu.tn/images/logo.png"
   
   par :
   
   src="assets/images/logo-iset-kairouan.png"

Le système utilise actuellement :
- URL principale : https://www.isetkr.rnu.tn/images/logo.png
- Fallback (si l'URL principale ne fonctionne pas) : Logo ISET générique de Wikipedia

Dimensions recommandées pour le logo :
- Format : PNG avec fond transparent
- Taille : 200x200 pixels minimum
- Ratio : Carré (1:1) de préférence
